# Agentic AI Internship Project

This repository contains a structured collection of Agentic AI development tasks completed as part of an internship program.  
It covers beginner to advanced level AI agent systems built using Python and Gemini, implementing agent-based architecture inspired by modern AI agent frameworks.

---

## 📌 Overview

The project is divided into multiple levels:

### 🟢 Beginner Level
- Tool-Calling AI Agent
- AI Calculator Agent

### 🟡 Intermediate Level
- Multi-Tool Agent
- RAG Assistant

### 🔴 Advanced Level
- Autonomous Business Agent
- Multi-Agent System
